## DhakaFlix Kodi Addon

**DhakaFlix** is a video add-on for Kodi that provides access to Bangladeshi content.

### üì• Installation Instructions (Install from ZIP File)

Follow these steps to install the DhakaFlix video add-on on your Kodi device.

#### 1. Enable Unknown Sources (If necessary)

Before installing a third-party add-on, you must enable "Unknown sources" in Kodi's settings.

1.  Open **Kodi**.
2.  Go to **Settings** (the gear icon ‚öôÔ∏è on the home screen).
3.  Select **System** (or **System settings** on older versions).
4.  Go to the **Add-ons** tab.
5.  Enable **Unknown sources**.
6.  Confirm the warning message by clicking **Yes**.

#### 2. Download the Add-on

Download the latest release of the DhakaFlix add-on ZIP file to a location on your device that you can easily access (e.g., your desktop or downloads folder).

**Latest Release Download Links (v1.0.0):**
* **Short URL:** [is.gd/dhkflx](https://is.gd/dhkflx)
* **Full URL:** [plugin.video.dhakaflix-1.0.0.zip](https://github.com/tazihad/dhakaflix-kodi/releases/download/v1.0.0/plugin.video.dhakaflix-1.0.0.zip)

#### 3. Install from ZIP File in Kodi

1.  Return to the Kodi home screen.
2.  Go to **Add-ons** from the main menu.
3.  Click the **Package installer icon** (the open box icon üì¶ at the top-left).
4.  Select **Install from zip file**.
5.  A file browser window will open. Navigate to the location where you saved the `plugin.video.dhakaflix-1.0.0.zip` file.
6.  Select the ZIP file.
7.  Wait for a moment. You will see a notification in the top-right corner that says **"DhakaFlix Add-on installed"**.

#### 4. Access the Add-on

The DhakaFlix video add-on is now installed and ready to use.

1.  From the Kodi home screen, select **Add-ons**.
2.  Go to **Video add-ons**.
3.  Select **DhakaFlix** to launch the add-on.
