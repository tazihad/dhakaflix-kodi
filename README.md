### ‚ö†Ô∏è Prerequisite: Enable Unknown Sources

Before installing, you must allow Kodi to load third-party add-ons.

1. Open **Kodi**.
2. Go to **Settings** (the gear icon ‚öôÔ∏è on the home screen).
3. Select **System** (or **System settings** on older versions).
4. Highlight **Add-ons** on the left menu.
5. Toggle **Unknown sources** to the **ON** position.
6. Click **Yes** on the warning popup.

### üì° Method 1: Install via File Manager (Recommended)

This method allows you to install the repository and addon directly through Kodi without downloading files manually.

#### Step 1: Add the File Source
1. Return to the Kodi home screen and click the **Settings** gear ‚öôÔ∏è.
2. Click on **File manager**.
3. Select **Add source**.
4. Click on the box that says **`<None>`**.
5. Type the following URL exactly as shown and click **OK**:
   > **`https://tazihad.github.io/dhakaflix-kodi/`**
6. Click the box under "Enter a name for this media source", type **`DhakaFlix`**, and click **OK**.

#### Step 2: Install the Repository ZIP
1. Go back to the Kodi home screen.
2. Select **Add-ons** from the left menu.
3. Click the **Package installer icon** (open box üì¶) at the top-left.
4. Select **Install from zip file**.
5. Find and select **`DhakaFlix`** from the list.
6. Click on **`repository.tazihad.dhakaflix-1.0.0.zip`**.
7. Wait for the **"DhakaFlix Repository Add-on installed"** notification in the top-right corner.

#### Step 3: Install DhakaFlix from Repository
1. While still on the same screen, click **Install from repository**.
2. Select **DhakaFlix Repository**.
3. Select **Video add-ons**.
4. Click on **DhakaFlix**.
5. Click **Install**.
6. Wait for the **"DhakaFlix Add-on installed"** notification.

### üíæ Method 2: Manual Install (Download ZIP)

Use this method if you prefer to download the file on a PC and transfer it via USB.

#### 1. Download the Repository
Download the latest repository ZIP file to your device.

**Latest Repository (v1.0.0):**
* **Direct Download:** [repository.tazihad.dhakaflix-1.0.0.zip](https://github.com/tazihad/dhakaflix-kodi/releases/download/v1.0.0/plugin.video.dhakaflix-1.0.0.zip)

#### 2. Install the Repo and Addon
1. From the Kodi home screen, go to **Add-ons**.
2. Click the **Package installer icon** (üì¶) at the top-left.
3. Select **Install from zip file**.
4. Navigate to the folder where you saved the ZIP file.
5. Select **`repository.tazihad.dhakaflix-1.0.0.zip`**.
6. Wait for the "Repository installed" notification.
7. Select **Install from repository** > **DhakaFlix Repository** > **Video add-ons** > **DhakaFlix** > **Install**.

3. Click **DhakaFlix** to start watching.
